# Task 1:Business Sales Dashboard from E-Commerce Dats
## Task Description
The objective of this task is to analyze e-commerce sales data to identify best-selling products, sales trends, and high-revenue categories. The analysis is performed using Power BI to create an interactive business sales dashboard that helps stakeholders make data-driven decisions.
## Approach
	1.	Imported the e-commerce dataset into Power BI.
	2.	Cleaned the data by handling missing values, correcting data types, and removing duplicates.
	3.	Performed data transformation using Power Query.
	4.	Created calculated measures using DAX for metrics such as total sales, revenue, and category-wise performance.
	5.	Designed interactive visuals including bar charts, line charts, and slicers to analyze sales trends and product performance.
	6.	Structured the dashboard to provide clear insights for business decision-making.
## Output
  •	An interactive Power BI dashboard displaying:
	•	Best-selling products
	•	Sales trends over time
	•	High-revenue product categories
	•	Key performance indicators (KPIs)
	•	The dashboard provides meaningful insights to support strategic business decisions.
