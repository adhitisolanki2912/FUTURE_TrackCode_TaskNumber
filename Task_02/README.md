# Task 2:Social Media Campaign Performance Tracker

## Task Description
The objective of this task is to analyze the impact of advertising spend across multiple marketing channels on product sales. Using Power BI, an interactive dashboard is created to evaluate channel-wise performance, total spend, sales trends, and return on investment (ROI).

## Tools Used
- Power BI Desktop
- CSV Dataset
- DAX (Data Analysis Expressions)
- Power Query

## Approach
1. Imported the advertising dataset into Power BI.
2. Cleaned and transformed data using Power Query.
3. Created calculated measures such as Total Ad Spend, Total Products Sold, Average Products Sold, and ROI using DAX.
4. Designed multiple visuals including bar charts, line charts, pie charts, KPI cards, and tables.
5. Added slicers to enable interactive analysis of advertising channels.
6. Structured the dashboard to provide clear business insights.

## Visuals Included
- KPI Cards (Total Spend, Total Sales, ROI)
- Bar Chart (Ad Spend by Channel)
- Line Chart (Products Sold Trend)
- Pie Chart (Ad Spend Contribution)
- Scatter Plot (Ad Spend vs Sales)
- Table (Detailed Campaign Performance)

## Output
An interactive Power BI dashboard that highlights the relationship between advertising investments and product sales, helping businesses identify high-performing marketing channels and optimize their advertising strategy.
